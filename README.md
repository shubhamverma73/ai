# 🚀 Learning AI in Node.js with LangChain, LangGraph, LangSmith & LangServe

This repository is a complete learning resource for building AI-powered applications using modern AI orchestration tools in **Node.js**.  
It includes practical examples, workflows, experiments, and integrations using:

- LangChain
- LangGraph
- LangSmith
- LangServe
- OpenAI APIs
- AI Agents & Workflows
- Basic Python examples for LangSmith

The goal of this repository is to help developers understand how to build scalable AI applications, AI agents, automation workflows, and LLM-powered systems using JavaScript/TypeScript.

---

# 📚 Technologies Covered

## 🔹 LangChain
Learn how to:
- Create LLM chains
- Work with prompts
- Use memory
- Build chatbots
- Create tools and agents
- Handle output parsers

---

## 🔹 LangGraph
Understand graph-based AI workflows:
- Multi-step AI agents
- Stateful workflows
- AI decision trees
- Conditional routing
- Agent orchestration
- Human-in-the-loop systems

---

## 🔹 LangSmith
Learn observability and debugging:
- Trace chains and agents
- Monitor token usage
- Debug workflows
- Analyze execution logs

Includes:
- Node.js examples
- One Python example file for LangSmith setup and tracing

---

## 🔹 LangServe
Deploy AI applications as APIs:
- Build REST APIs for AI workflows
